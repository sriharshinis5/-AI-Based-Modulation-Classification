# AI-Based Modulation Classification Using Machine Learning

## Overview

This project classifies digital modulation schemes using machine learning techniques.

Supported Modulations:

* BPSK
* QPSK
* 16-QAM
* 64-QAM

The model is trained using generated signal features and predicts the modulation type automatically.

## Technologies Used

* Python
* NumPy
* Pandas
* Scikit-Learn
* Machine Learning

## Applications

* Cognitive Radio
* Wireless Communication Systems
* Adaptive Modulation
* Signal Intelligence

## Project Workflow

1. Generate signal dataset
2. Train Random Forest Classifier
3. Evaluate model accuracy
4. Predict modulation scheme

## Results

Classification Accuracy: 95%+
